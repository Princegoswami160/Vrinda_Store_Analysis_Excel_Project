# Vrinda Store Sales Analysis (Excel Dashboard Project)

## Project Overview
This project focuses on analyzing the **Vrinda Store Dataset (31,048 rows)** to generate meaningful business insights and build an **interactive Excel dashboard**.

The main objective is to understand customer behavior, sales trends, and channel performance to help the store improve sales strategies for **2025**.

---

##  Objective
To create an **Annual Sales Report for 2024** that helps:
- Understand customer demographics  
- Identify top-performing states and channels  
- Analyze sales trends and order patterns  
- Provide actionable insights to increase future sales  

---

##  Dataset Information

- **Dataset Name:** Vrinda Store Dataset  
- **Total Rows:** 31,048  
- **Format:** Excel (.xlsx)

###  Columns Used:
    index, Order ID, Cust ID, Gender, Age, Age Group, Date, Month,
    Status, Channel, SKU, Category, Size, Qty, currency, Amount,
    ship-city, ship-state, ship-postal-code, ship-country, B2B

##  Dashboard Features

##  Key Insights

1. **Women contribute ~60–65% of total sales**  
2. **Top performing states:** West Bengal, Uttar Pradesh, Telangana  
3. **Age group 30–49 years** contributes the highest orders  
4. **Top channels:** Amazon, Flipkart, Myntra (~80% contribution)  
5. Majority orders are **successfully delivered (~90%+)**

---

## 📌 Final Conclusion

To improve sales:
- Target **Women customers (Age 30–49)**  
- Focus on **Top states (West Bengal, UP, Telangana)**  
- Run ads, offers, and coupons on:
  - Amazon  
  - Flipkart  
  - Myntra
